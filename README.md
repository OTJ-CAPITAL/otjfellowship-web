# otjfellowship.com

Website for **OTJ Fellowship** — a selective 12-week program developing the next generation of African quantitative researchers, engineers, and fund managers.

---

## What this site is

The application and information hub for the fellowship program. Fellows who complete the program work directly on OTJ Capital — building real strategies, deploying real systems, on a real fund.

**Sections:**
- Hero — positioning + stats (10 fellows, 12 weeks, paid stipend, Q3 2026 start)
- About — why Africa needs its own quants
- Program — three tracks: Quant Research · Engineering · Fund Operations
- Apply — inline application form (name, email, track, motivation)

---

## Program tracks

| Track | Duration | Focus |
|---|---|---|
| Quant Research | 12 weeks | Alpha strategy design, backtesting, live deployment |
| Engineering | 12 weeks | Trading systems, data pipelines, risk engine, infra |
| Fund Operations | 8 weeks | Portfolio construction, risk management, investor reporting |

Top fellows join OTJ Capital full-time.

## Development

```bash
npm install
npm run dev
# → http://localhost:3000
```

---

## Deploy

1. Connect `OTJ-CAPITAL/otjfellowship-web` to Vercel
2. No environment variables required
3. Set custom domain: `otjfellowship.com`
4. Every push to `main` auto-deploys
